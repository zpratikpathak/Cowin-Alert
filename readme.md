## Installation
`pip install -r requirements.txt`

Create a `.env` file in the root directory and add the following credentials

```
PINCODES=
smtpemail = 
smtpPassword = 
receiverEmail = 
```

- Add pincodes separated by Space eg. `PINCODES=12345 12343 12343`
## How to run?
Use command

```python tracker .py```
